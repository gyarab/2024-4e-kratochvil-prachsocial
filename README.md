# Prachsocial

## Installation
```bash
npm i --legacy-peer-deps
```

### Create .env file
```bash
POSTGRES_USER="username"
POSTGRES_HOST="host"
POSTGRES_PASSWORD="password"
POSTGRES_DATABASE="database"
```

### Setup Prisma
```bash
npx prisma generate
```

## Running Development Server
```bash
npm run dev 
```
